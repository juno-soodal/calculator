## 2025.01.02 개발 기능
### Lv 1. 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기

- [ ]  **양의 정수(0 포함)를 입력받기**
- scanner를 사용하여 숫자 입력 받기
- 문자 입력 예외 처리
- [ ]  **사칙연산 기호(➕,➖,✖️,➗)를 입력받기**
- scanner를 사용하여 숫자 입력 받기
- (➕,➖,✖️,➗) 이외의 문자 입력 예외 처리
- [ ]  **위에서 입력받은 양의 정수 2개와 사칙연산 기호를 사용하여 연산을 진행한 후 결과값을 출력하기**
- 1 + 2 = 3 형식으로 출력
- [ ]  **반복문을 사용하되, 반복의 종료를 알려주는 “exit” 문자열을 입력하기 전까지 무한으로 계산을 진행할 수 있도록 소스 코드를 수정하기**
- scanner를 사용하여 문자 입력 받기

## 이슈
 - 문제: 문자를 입력하니 InputMismatchException 발생
 - 해결  정상적인 숫자입력할때까지 while문 안에 넣기  정상적인 값인지 확인 하기 위해 try catch로 성공시 break 탈출

 - 문제: 문자 입력시 catch문까지 오는건 성공했지만 무한 반복하는 오류 발생
 - 해결: 에러 메시지 출력후 끝났다고 알려주기 위해 nextLine() 추가
