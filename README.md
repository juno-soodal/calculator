프로젝트 설명
사용자가 입력한 두 숫자와 연산자를 기반으로 산술연산을 수행합니다.
연산기호는 +, -, *, / 4가지 있습니다.
예외 처리 및 명령어 기반의 추가 기능도 있습니다.

기능
사칙연산
명령어 처리:exit
예외 처리: 문자 입력 시, 0으로 나누는 경우, 없는 연산자

사용 방법

프로그램 실행 후, 다음과 같은 형식으로 진행됩니다.
1.첫 번째 숫자를 입력합니다.
2.두 번째 숫자를 입력합니다.
3.연산자를 입력합니다.
4.계한 결과가 출력됩니다.
5.계속해서 진행할지 명령어를 입력합니다.

코드 구조
CalcualtorApp:  사용자 입력, 계산 및 결과 출력, 사용자 명령어 처리하는 클래스
